## 📖 O que é o IMPERIUS?

**IMPERIUS** é um mundo paralelo onde guerreiros, magos e assassinos vivem e morrem pela glória.

Aqui não existem leis. Existe apenas poder.
Quem é forte, sobrevive. Quem é fraco, perece.

Um bot de RPG completo para WhatsApp — com batalhas épicas, 21 regiões para explorar, pets míticos para domar, guildas para liderar, e o próprio Deus para desafiar.

## ⚔️ Sistemas Principais

### 🎭 Classes
- **20 classes normais** — Guerreiro, Mago, Assassino, Necromante e muito mais
- **11 classes raras** — obtidas pela Roleta do Destino
- **👑 Ajudante do Deus** — a classe mais rara do IMPERIUS, porta a Lâmina Primeva e um Ovo do Dragão de Deus

### ⚔️ Batalha
- 🎲 Sistema D20 — o dado decide o destino de cada ataque
- Batalhas interativas por turnos
- Pets que auxiliam e amplificam golpes
- Bosses com múltiplas fases
- PvP com turnos alternados entre jogadores

### 🐾 Pets Míticos
- **46+ criaturas** para domar em batalha
- Sistema de estados: 😴 Dormindo → ☠️ Possesso
- Cada pet é **ÚNICO** no servidor
- Formato: **Espécie (Nome)**

### 🗺️ Exploração
- Viaje entre 21 regiões únicas
- Encontre criaturas raras pelo caminho
- Descubra masmorras escondidas

### 💰 Economia
- Sistema de moedas próprio
- Loja completa de itens e armas
- **20 raridades** de armas — de Comum até ☠️ DEUS
- Moeda oficial: **Primela Barius** (Belarium) 🪙
- Doe Belariums para outros jogadores

### ☠️ O Deus
- **JUVENT** — o Deus Supremo e Arquiteto do Caos
- Eventos divinos que abalam o servidor inteiro
- Apenas o **Ajudante do Deus** pode chegar perto dele

## 🏆 Como Jogar

```
1️⃣ Entre em um grupo com o bot
2️⃣ Digite /criar para criar seu personagem
3️⃣ Tente a /roleta para uma classe rara
4️⃣ Explore o mundo com /viajar
5️⃣ Batalhe com /batalha
6️⃣ Evolua ou morra.
```
